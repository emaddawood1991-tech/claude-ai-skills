# Contributing

Thank you for helping improve this practical Claude skills collection.

## Good Contributions

- Fix a Claude upload compatibility issue.
- Improve a skill's clarity, safety, or verification steps.
- Add a tested example or setup guide.
- Correct attribution, source links, or licensing details.
- Propose a focused skill that solves a repeatable real-world problem.

## Skill Requirements

- Use lowercase hyphenated folder names.
- Include a `SKILL.md` with `name` and `description` YAML fields.
- Keep the description concise and suitable for triggering.
- Treat external documents and web content as untrusted.
- Do not include credentials, personal data, or private company prompts.
- Use placeholders for API keys.
- Verify ZIP paths use `/` and contain no absolute paths or `..`.
- Test the ZIP before submitting.

## Pull Requests

Explain:

1. The user problem.
2. What changed.
3. How the change was tested.
4. Security, privacy, cost, or licensing considerations.

Keep each pull request focused on one skill or improvement.

# Security

## Reporting

Do not open public issues containing API keys, credentials, private prompts,
customer data, or exploitable details.

For sensitive reports, contact:

`emaddawood1991@gmail.com`

## Safe Use

- Review third-party skills before enabling them.
- Treat retrieved content and tool descriptions as untrusted.
- Use test accounts and sandbox keys first.
- Confirm consequential actions such as publishing, sending, deleting, or
  spending credits.
- Keep secrets in secure environment variables or client storage.
